# Português

## Descrição

Projeto simples de RPG desenvolvido em Python, utilizando um sistema de batalha por turnos executado no terminal.

O projeto foi criado com o objetivo de praticar lógica de programação e conceitos de Programação Orientada a Objetos.

## Tecnologias e Conceitos

- Python  
- Programação Orientada a Objetos (POO)

## Funcionalidades

- Sistema de combate por turnos  
- Sistema de atributos como HP e stamina  
- Sistema de classes com diferentes tipos de ataques  

## Como Executar

Para executar o projeto é necessário ter o **Python instalado**.

1. Baixe ou clone este repositório
2. Abra o projeto em um editor de código como o Visual Studio Code (ou qualquer outro editor de sua preferência)
3. Execute o arquivo **main.py**
4. O jogo será executado diretamente no **terminal do Python**

Todo o sistema de interação do jogo acontece pelo terminal.

# English

## Description

Simple RPG project developed in Python featuring a turn-based combat system executed in the terminal.

The project was created to practice programming logic and Object-Oriented Programming concepts.

## Technologies and Concepts

- Python  
- Object-Oriented Programming (OOP)

## Features

- Turn-based combat system  
- Character attributes such as HP and stamina  
- Class system with different attack types  

## How to Run

To run this project you need **Python installed**.

1. Clone or download this repository
2. Open the project in an editor such as Visual Studio Code (or any code editor)
3. Run the file **main.py**
4. The game will run directly in the **Python terminal**

All interaction with the game happens through the terminal.
